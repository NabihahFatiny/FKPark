# FK PARK- Car Parking Management System
FKPark is a web-based Car Parking Management System built with PHP and MySQL, featuring vehicle registration, parking slot booking, QR code-based access control, and traffic summons management for efficient campus parking operations.

## Features

- User authentication with role-based access (Student, Admin, Security)
- Vehicle registration and approval system
- QR code generation for vehicle and parking verification
- Parking slot management and real-time availability
- Parking booking system with create, edit, and cancel functionality
- Traffic summons management and violation tracking
- Admin dashboard with reports and analytics
- Secure database integration using MySQL
